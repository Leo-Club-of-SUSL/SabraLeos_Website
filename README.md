# 🦁 Leo Club of SUSL - Official Website

Welcome to the digital home of the Leo Club of Sabaragamuwa University of Sri Lanka (SUSL). This project is a modern, high-performance web platform designed to showcase our club's impact and activities to the world.

---

## 🚀 Quick Start
To get the project running locally:

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Environment Variables**:
    Create a `.env` file based on `.env.example`.
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Deployment**:
    Hosted on **Cloudflare Pages**. Builds are automatically triggered via Git.

## 🏗 Features
- **Modern CMS**: Manage content directly via a secure Admin Dashboard.
- **Dynamic Media**: High-performance image loading using Cloudinary & Supabase.
- **Secure Auth**: Admin-only access with security monitoring and brute-force protection.
- **WCAG AA Compliant**: Optimized for accessibility and screen readers.
- **Mobile First**: Fully responsive design for all devices.
